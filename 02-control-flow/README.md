# Control Flow

## 📖 Overview

Control Flow determines the order in which a program executes its statements. Instead of executing every line sequentially, Python allows programs to make decisions, repeat tasks, and control execution based on conditions.

Control Flow is one of the fundamental concepts in programming and forms the basis of problem-solving and algorithm design.

---

## ❓ Why Do We Need Control Flow?

Real-world programs rarely execute every instruction one after another.

For example:

- A login system checks whether the password is correct.
- An ATM verifies whether the account has sufficient balance.
- A chatbot decides how to respond based on the user's input.
- An AI model repeats training for thousands of iterations.

These situations require programs to make decisions and repeat actions, which is made possible through Control Flow.

## 🤖 AI Connection

Control Flow is used throughout AI applications.

Examples include:

- Repeating model training using loops.
- Making predictions based on conditions.
- Processing datasets one record at a time.
- Validating input before making predictions.
- Controlling the execution of AI pipelines.

Without Control Flow, programs cannot make decisions or automate repetitive tasks.
