# Data Types

## 📖 Overview

Every piece of data in Python has a **data type**. A data type tells Python what kind of value it is working with and determines how that value can be stored, processed, and manipulated.

Whether you're storing a person's age, name, salary, or login status, Python first identifies the type of that data before performing any operation on it.

Understanding data types is one of the most important foundations of programming.

---

## ❓ Why Do We Need Data Types?

Different kinds of data behave differently.

For example:

- Numbers can be added or multiplied.
- Text can be joined together.
- Boolean values can be used to make decisions.

Without data types, Python would not know how to process different kinds of information correctly.

---

## 🧠 Data Types Covered

This section covers Python's fundamental built-in data types:

- Integer (`int`)
- Float (`float`)
- String (`str`)
- Boolean (`bool`)

You will also learn how to identify the type of a value using the built-in `type()` function.

---

## 🤖 AI Connection

Artificial Intelligence systems work with many different types of data.

Examples include:

- Images
- Text
- Numbers
- Labels
- Predictions
- Boolean conditions

Before an AI model can process information, Python must correctly identify and manage the data type being used.

A strong understanding of data types helps prevent bugs and ensures data is processed correctly.
