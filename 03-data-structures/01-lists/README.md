# Lists

## 📖 Overview

A list is one of Python's most versatile and widely used built-in data structures. It stores multiple values in a single variable while preserving their order.

Lists are mutable, meaning their contents can be modified after creation by adding, removing, or updating elements.

Because of their flexibility, lists are one of the most frequently used data structures in Python and play a significant role in Data Science, Machine Learning, and Artificial Intelligence.

---

## ❓ Why Do We Need Lists?

Imagine storing the marks of 500 students.

Without lists:

```python
student1 = 95
student2 = 88
student3 = 76
...
student500 = 91
```

Managing hundreds of variables quickly becomes inefficient.

Using a list:

```python
marks = [95, 88, 76, ..., 91]
```

Now all related data is stored together, making it easier to access, update, process, and analyze.

## 🤖 AI Connection

Lists are often the first structure used to collect and organize data before it is processed by AI libraries such as NumPy and Pandas.

Common AI use cases include:

- Storing raw datasets
- Collecting user inputs
- Managing predictions
- Organizing labels
- Preparing data before model training

Understanding lists builds the foundation for working with larger and more efficient data structures in AI.
